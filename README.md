# Fehla RPG
My very first C# project ever is a RPG prototype with a unique mechanical twist and a cell-based canvas renderer, all inside a csharp-console.

## This was a graded project in Retraining where I received a 98% total score.
We were given 7 days to code a csharp project of our choice, document it and present it on day 8. It was allowed to use AI assistance, but as it was my first project I used a Teacher AI to teach me concepts and figure out solutions myself. The "Fehla" project originated from a little fictional universe I wrote lore for in January 2025, drew a character design and made a 2D virtual model of. I chose this as the lore of this "Fehlaverse" is all about breaking the perception of the observer, telling weird and fun stories with unexpected twists on the meta-level.

## Features:
- Title menu with colored ASCII
- Prototype Encounter against a Frog
- Updating game HUD including menu
- Encounter indicators and attack patterns
- Scrolling text box with confirm prompt
- Game over screen (currently no win condition)

## Code Implementation:
- Csharp console canvas renderer updater
- Methods for writing symbols into cells
- Methods for coloring specific cells and areas
- Basic game loop (only one test-encounter)
